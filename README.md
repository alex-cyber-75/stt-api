# STT-API

This repository contains the source code for the Speech‑to‑Text API.

## Features
- Speech‑to‑Text API powered by Whisper.Net
- Minimal F# ASP.NET Core Web API
- Exposes `/transcribe` endpoint for audio uploads

## Getting Started
1. Clone the repository.
2. Restore NuGet packages: `dotnet restore`.
3. Build and run: `dotnet run --project STTApi.fsproj`.
